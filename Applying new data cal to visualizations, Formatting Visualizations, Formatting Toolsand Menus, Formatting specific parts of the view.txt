1. Create Bar Chart
	i) Drag a "Region" to Columns.
	ii) Drag a "Profit" to Rows.

2. Format Text and Labels
	i) Right-click axis or labels → select Format.
	ii) Change font, size, color (e.g., bold, italic for region names).

3. Format Chart Marks (Bars, Lines, etc.)
	i) Click Marks Card → choose mark type (e.g., bar, area).
	ii) Use Color, Size, Label, Tooltip to modify appearance.

4. Customize Chart Title and Axis
	i) Double-click title → edit font, size, color.
	ii) Right-click axis → Edit Axis or use Format for labels
