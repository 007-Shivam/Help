1. Edit Axis Title and Labels
	i) Click axis title → edit font, size, color.
	ii) Right-click axis → Edit Axis → adjust scale, min/max values, tick marks.

2. Change Data Type
	i) In the Data Pane, click the icon next to "Order ID" → choose change data type.

3. Create “DATE DIFFERENCE” Field
	i) Formula: DATEDIFF('day', [Order Date], [Ship Date])
	ii) Drag it to Text on the Marks card.
	Right-click → Measure > Average.

4. Other Date Functions
	i) DATEADD('day', 5, [Order Date])
	ii) DATENAME('month', [Order Date])
	iii) MAKEDATE(2023, 6, 2)
	iv) MAKEDATETIME([Order Date], #12:00:00#)

5. Display Results in table or chart using Text, Rows, or Columns shelf.
