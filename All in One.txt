==========================Exp 4=======================================
Create various types of graphs and charts in tableau

1. Create Bar Chart
	i) Drag a "Region" to Columns.
	ii) Drag a "Profit" to Rows.

2. Create Stacked Bar Chart
	i) Repeat Bar Chart steps.
	ii) Drag "Ship Mode" field to the “Color” shelf on the Marks card.

3. Create Line Chart (Optional)
	i) Drag a date field to Columns and a measure to Rows.
	ii) Select “Line” from the Marks card.

==========================Exp 5=======================================
Tableau Cal. SUM, AVR, and Agg features, Creating custom cal and fields

1. Use SUM Function
	i) Drag a "Profit" to Rows or Columns shelf.

2. Use AVG Function
	i) Drag a "Profit" and switch aggregation to Average

3. Use Other Aggregate Functions
	i) COUNT: COUNT([FieldName])
	ii) MIN: MIN([FieldName])
	iii) MAX: MAX([FieldName])
	iv) COUNTD: COUNTD([FieldName])

4. Create Custom Calculated Fields
	i) Go to the top menu > Analysis > Create Calculated Field 
	ii) Create Calculated Field.
	iii) Enter formula (e.g., [Sales] - [Profit]) and name it "Formula1".
	iv) Click OK.

5. Drag Custom Fields into your worksheet to use them in charts.

==========================Exp 6=======================================
Applying new data cal to visualizations, Formatting Visualizations, Formatting Toolsand Menus, Formatting specific parts of the view

1. Create Bar Chart
	i) Drag a "Region" to Columns.
	ii) Drag a "Profit" to Rows.

2. Format Text and Labels
	i) Right-click axis or labels → select Format.
	ii) Change font, size, color (e.g., bold, italic for region names).

3. Format Chart Marks (Bars, Lines, etc.)
	i) Click Marks Card → choose mark type (e.g., bar, area).
	ii) Use Color, Size, Label, Tooltip to modify appearance.

4. Customize Chart Title and Axis
	i) Double-click title → edit font, size, color.
	ii) Right-click axis → Edit Axis or use Format for labels

==========================Exp 7=======================================
Editing and Formatting Axes, Manipulating Data in Tableau data, Pivoting Tableau data, Date functions

1. Edit Axis Title and Labels
	i) Click axis title → edit font, size, color.
	ii) Right-click axis → Edit Axis → adjust scale, min/max values, tick marks.

2. Change Data Type
	i) In the Data Pane, click the icon next to "Order ID" → choose change data type.

3. Create “DATE DIFFERENCE” Field
	i) Formula: DATEDIFF('day', [Order Date], [Ship Date])
	ii) Drag it to Text on the Marks card.
	Right-click → Measure > Average.

4. Other Date Functions
	i) DATEADD('day', 5, [Order Date])
	ii) DATENAME('month', [Order Date])
	iii) MAKEDATE(2023, 6, 2)
	iv) MAKEDATETIME([Order Date], #12:00:00#)

5. Display Results in table or chart using Text, Rows, or Columns shelf.

==========================Exp 9=======================================
Find frequency using Word cloud and bar graph

1. Create a Word Cloud
	i) Drag Customer Name to the Text card
	ii) Drag Region to Filter and select West.
	iii) Drag while holding the right mouse button Customer Name and drop it on Size. Then select CNT(Customer Name)
	iv) changing the mark type to Text.
	v) dragging Customer Name to Filters and click the “Top” tab. Select By field and specify Top 20 by Customer Name (Count)
	vi) drag while holding Control CNT(Customer Name) to the Colour card

2. Create a Bar Graph
	i) Drag the text field to Rows.
	ii) Drag it again to Columns, but set aggregation to Count or CountD.
	iii) Sort bars descending by frequency.
	iv) Format labels, colors, and axes for clarity.
