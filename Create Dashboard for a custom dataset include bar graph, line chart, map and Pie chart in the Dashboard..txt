1. Create Bar Chart
	i) Drag a "Region" to Columns.
	ii) Drag a "Profit" to Rows.

2. Create Stacked Bar Chart
	i) Repeat Bar Chart steps.
	ii) Drag "Ship Mode" field to the “Color” shelf on the Marks card.

3. Build Dashboard
	i) Click New Dashboard (bottom tab).
	ii) Drag created charts (bar, line, pie, map) into layout.
